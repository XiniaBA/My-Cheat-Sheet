My Cheat Sheet for HTML tables

Table tags

Element Tag Definition
Table <table> The wrapper element for all HTML tables.
Table head <thead> The set of rows defining the column headers in a table.
Table body <tbody> The set of rows containing actual table data.
Table footer <tfoot> The set of rows defining the footer in a table.
Table row <tr> The table row container.
Table heading <th> To recognize the table headers as headers, both visually and semantically.
Table data <td> The table row container.
